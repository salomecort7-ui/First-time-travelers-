# FirstJourney 🗺️

Your first vacation, made simple.

A vacation planning service built for absolute first-time travelers. We handle everything from destination suggestions to booking to answering every "dumb question" — so you get a stress-free, affordable trip with great reviews, not surprises.

## Built With

- Vite + React
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Live Site

The production build is in `/dist`.
